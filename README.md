# Flowing Loop

다시 흐름으로 돌아오는 시작페이지

## 개념

- 목표를 관리하거나 평가하지 않습니다
- 하루의 흐름이 끊겼을 때 다시 루프로 돌아오게 돕는 앱
- 실패, 미달성, KPI 같은 단어는 사용하지 않습니다
- 오늘만 존재하는 구조

## 특징

- 하루 최대 3개의 태스크
- Done / Not today 두 가지 액션
- 부드러운 미니멀 디자인
- 죄책감 유발 요소 제거
- 매일 자정에 자동 리셋

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 디자인 철학

- Soft minimal
- Calm productivity
- Emotional friction 최소화
- Microsoft To Do보다 부드럽고, Notion보다 가볍게

## 기술 스택

- React 18
- TypeScript
- Vite
- Pure CSS (no frameworks)
